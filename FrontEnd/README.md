# Minha Aula - Sistema de Gerenciamento de Aulas

Sistema completo para gerenciamento de aulas particulares, desenvolvido com Next.js e otimizado para dispositivos móveis.

## Funcionalidades

- 📚 Gerenciamento de alunos
- 📅 Agenda de aulas
- 💰 Controle financeiro
- 📊 Dashboard com estatísticas
- 🌙 Modo escuro/claro
- 🌍 Suporte a múltiplos idiomas (PT/EN)
- 📱 Interface otimizada para mobile

## Tecnologias

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Radix UI
- Local Storage para persistência

## Como usar

1. Clone o repositório
2. Instale as dependências: `npm install`
3. Execute em desenvolvimento: `npm run dev`
4. Acesse: `http://localhost:3000`

## Deploy

O aplicativo está configurado para deploy automático e pode ser acessado em produção.

## Estrutura

- `/app` - Páginas e layouts
- `/components` - Componentes reutilizáveis
- `/hooks` - Hooks customizados
- `/lib` - Utilitários e configurações
- `/public` - Arquivos estáticos
